# REAM Technical Test (React + Node.js)

## How to Run

### Backend
1. `cd server`
2. `npm install`
3. `npm run dev` (or `node index.js`)
4. Ensure your DB connection string is updated in `.env`

### Frontend
1. `cd client`
2. `npm install`
3. `npm start`

### Database
- Run the scripts in `db/assets.sql` using SQL Server Management Studio
